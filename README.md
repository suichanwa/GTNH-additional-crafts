# GTNH Additional Crafts

A GregTech New Horizons mod that adds additional crafting recipes.

## Features

- Custom GregTech recipes
- Integration with existing GTNH items

## Development

This mod is based on the [GTNH ExampleMod](https://github.com/GTNewHorizons/ExampleMod1.7.10) template.

### Building

```
./gradlew build
```
